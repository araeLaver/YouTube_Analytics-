1️⃣ 채널 기초 설정 (Day 1-2)

□ YouTube Studio 접속 → 설정 → 채널 → 기본정보

□ 시각적 브랜딩
  ✓ 채널 아트 제작 (2560x1440px, Canva 무료 템플릿 활용)
  ✓ 프로필 사진 설정 (800x800px)
  ✓ 워터마크 추가 (구독 유도용)

□ 재생목록 구성
  ✓ 카테고리별 재생목록 3-5개 생성
  ✓ 인기 동영상 재생목록
  ✓ 신규 방문자용 소개 재생목록

  ✓ 시청 시간대 파악
  ✓ 선호 콘텐츠 길이 결정

□ 콘텐츠 캘린더 작성
  ✓ 월간 주제 설정
  ✓ 주간 업로드 일정 (최소 주 2회)
  ✓ 특별 이벤트/기념일 콘텐츠 계획

□ 키워드 리서치
  ✓ Google Trends 활용
  ✓ YouTube 자동완성 키워드 수집
  ✓ 경쟁 채널 인기 영상 키워드 분석

3️⃣ 첫 콘텐츠 제작 (Day 5-7)
───────────────────────────────────────────────────────────────────────
□ 영상 제작
  ✓ 스크립트 작성 (후킹-본론-CTA 구조)
  ✓ 촬영 (좋은 조명 + 깨끗한 오디오)
  ✓ 편집 (처음 15초 집중)

□ 최적화
  ✓ 제목: 주요 키워드 앞쪽 배치 (60자 이내)
  ✓ 설명: 첫 125자 핵심 내용, 총 500자 이상
  ✓ 태그: 관련 태그 15-20개
  ✓ 썸네일: 고대비, 큰 텍스트, 얼굴 표정

□ 업로드 설정
  ✓ 종료화면 추가 (마지막 20초)
  ✓ 카드 삽입 (관련 영상 추천)
  ✓ 자막 추가 (자동 생성 후 수정)
